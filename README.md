# Trabalho-IA-DeepLearning
Trabalho da disciplina de inteligência artificial, focado em ensinar o funcionamento das redes neurais convolucionais(CNN) através da classificação de imagens classificadas previamente pelo Keras em três categorias: Cachorros, gatos e vida selvagem, buscando alcançar a maior eficácia possível e apontar os resultados com os possíveis acertos e erros do modelo.
